Telegram ChatGPT Bot

Описание проекта

Telegram-бот, интегрированный с OpenAI ChatGPT, который предоставляет интеллектуальные ответы на вопросы пользователей с сохранением контекста диалога.

Выполненные требования

Основные функции

1.Команды бота

  /start - инициализация бота и начало нового диалога
  
  /help - справка по использованию бота
  
  "Новый запрос" - кнопка для сброса истории диалога

2.Интеграция с ChatGPT API

  Отправка пользовательских запросов в OpenAI GPT-3.5 Turbo
  
  Получение и отображение интеллектуальных ответов

3.Система истории диалогов

  Сохранение контекста для каждого пользователя отдельно
  
  Использование предыдущих сообщений для улучшения качества ответов
  
  Автоматический сброс истории при команде /start или нажатии "Новый запрос"

4.Пользовательский интерфейс

  Кастомная клавиатура для удобного взаимодействия
  
  Индикатор набора сообщения (typing)
  
  Обработка ошибок с информативными сообщениями

Техническая реализация

1.Используемые технологии

  Python 3.11+
  
  pyTelegramBotAPI - для работы с Telegram Bot API
  
  OpenAI Python Client - для взаимодействия с ChatGPT API

2.Архитектура

  Глобальный словарь user_history для хранения истории по user_id
  
  Системные промпты для настройки поведения ассистента
  
  Обработка исключений при работе с внешними API


Проблемы и решения

Проблема 1: Географические ограничения

Описание:

OpenAI API недоступен в некоторых странах

Решение: 

Использование VPN

Проблема 2: Финансовые ограничения

Описание:

Необходимость оплаты ChatGPT API

Отсутствие работающих бесплатных токенов OpenAI

Решение:

Не нашлось)

Проблема 3: Тестирование функциональности

Описание:

Невозможность полноценного тестирования из-за отсутствия рабочего API-ключа


Установка и запуск

Установите зависимости:

bash

pip install pytelegrambotapi openai

Настройте токены:

python

bot = telebot.TeleBot('YOUR_TELEGRAM_BOT_TOKEN')

client = OpenAI(api_key='YOUR_OPENAI_API_KEY')

Запустите бота:

bash

python main.py
